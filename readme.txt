== Description ==

Plugin for WordPress to insert the code snippet for [IE6Update](http://IE6Update.com) into your WordPress pages.
Read more about the IE6Update code snippet here: [Announcing IE6 Update - Help Kill Internet Explorer 6](http://almost.done21.com/2009/04/announcing-ie6-update-help-kill-internet-explorer-6/)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `wp-ie6update` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
